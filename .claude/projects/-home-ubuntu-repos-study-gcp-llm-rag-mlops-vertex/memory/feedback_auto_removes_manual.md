---
name: 自動化したら手順書から消す
description: 手動手順を自動化した場合、その手順セクションは運用ドキュメントから削除する。自動化されたものをユーザーが手動実行することはないため。
type: feedback
---

自動化した手順は運用ドキュメントから削除する。「自動実行される。手動でも可能」のような記述は残さない。

**Why:** ユーザーが実行することのない手順を残すのは矛盾であり、ドキュメントのノイズになる。
**How to apply:** 手動ステップを Makefile 等に自動化した際、対応する手順セクションは丸ごと削除する。ステップ一覧表の1行で自動化されていることが分かれば十分。
