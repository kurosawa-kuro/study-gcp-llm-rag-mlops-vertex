## ディレクトリ構成

```
study-gcp-llm-rag-mlops-vertex/
├── .github/workflows/      # CI/CD（GitHub Actions）
├── data/sample/            # サンプル社内規定ドキュメント
├── docs/                   # 仕様・設計書
├── env/
│   ├── config/             # プロジェクト設定（application.yml）
│   └── secret/             # クレデンシャル（git管理外）
├── makefiles/              # Makefile分割モジュール
├── scripts/                # 共通設定・ユーティリティ・監視・ES操作
├── src/
│   ├── doc-qa/
│   │   ├── ingestion/      # Cloud Run Job（ドキュメント取込）
│   │   └── api/            # Cloud Run Service（QA API）
│   └── elastic-search/     # Elastic Cloud接続確認PoC
└── terraform/
    └── modules/            # data / compute / elastic / registry / iam
```

## ファイル一覧

### ルート

| ファイル | 説明 |
|---------|------|
| `.dockerignore` | Docker build context 除外設定 |
| `.env.example` | 環境変数テンプレート + シークレット管理ガイド |
| `.gitignore` | Git除外設定 |
| `CLAUDE.md` | Claude Code向けプロジェクト説明書 |
| `Makefile` | ビルド・デプロイコマンド統合エントリポイント |
| `README.md` | プロジェクト概要 |

### .github/workflows/

| ファイル | 説明 |
|---------|------|
| `doc-qa.yml` | Ingestion + API CI/CD（test→build→push→deploy） |
| `terraform.yml` | Terraform CI/CD（PR→plan表示、master→apply自動実行） |

### env/

| ファイル | 説明 |
|---------|------|
| `config/application.yml` | プロジェクト固有設定（全設定の唯一の定義元） |
| `secret/credentials.yml` | 全クレデンシャル統合（GCP SA + Elastic Cloud）※git管理外 |

| `es_config.py` | Elastic Cloud 接続確認用の共通設定 |
| `es_docker_ops.py` | Elastic Cloud 接続確認用 Docker操作 |
| `es_gcp_ops.py` | Elastic Cloud 接続確認用 GCP操作 |
| `es_tf_ops.py` | Elastic Cloud 関連 Terraform操作 |
| `monitor_ingestion.py` | Ingestion Job実行結果監視 + Discord通知 |
| `monitor_qa_api.py` | QA API健全性チェック + Discord通知 |
| `upload_doc.sh` | GCSへのドキュメントアップロード |
| `query.sh` | QA APIへのクエリテスト |
| `setup_gcp.sh` | Google Cloud CLIインストール・初期設定 |
| `setup_terraform.sh` | Terraformインストール・初期設定 |

### src/doc-qa/ingestion/（ドキュメント取込パイプライン）

| ファイル | 説明 |
|---------|------|
| `main.py` | オーケストレーション（config.py 経由で全設定取得） |
| `extractor.py` | テキスト抽出（PDF: pymupdf / Word: python-docx / TXT） |
| `chunker.py` | チャンク分割（引数で chunk_size/overlap を受け取る） |
| `embedder.py` | Vertex AI Embedding API（引数で model_name/batch_size を受け取る） |
| `bq_writer.py` | BigQuery documents テーブル書き込み（冪等・リトライ付き） |
| `es_writer.py` | Elasticsearch kuromoji インデックス登録（引数で index_name を受け取る） |
| `Dockerfile` | build context=ルート、config.py + application.yml を含む |
| `requirements.txt` | 依存パッケージ（PyYAML含む） |
| `requirements-dev.txt` | 開発依存（pytest, pytest-mock） |
| `test_main.py` | 単体テスト（extractor, chunker, bq_writer） |

### src/doc-qa/api/（QA API）

| ファイル | 説明 |
|---------|------|
| `main.py` | FastAPIエントリポイント（config.py 経由で全設定取得） |
| `embedder.py` | クエリEmbedding（config.py からモデル名取得） |
| `retriever.py` | ハイブリッド検索（BigQuery Vector Search + Elasticsearch kuromoji） |
| `reranker.py` | RRF リランク（k は呼び出し元から引数で受け取る） |
| `generator.py` | Vertex AI Gemini回答生成（config.py からモデル名取得） |
| `Dockerfile` | build context=ルート、config.py + application.yml を含む |
| `requirements.txt` | 依存パッケージ（PyYAML含む） |
| `requirements-dev.txt` | 開発依存（pytest, httpx, pytest-mock） |
| `test_main.py` | 単体テスト（reranker, generator） |

### terraform/（モジュール分離構成）

| ファイル | 説明 |
|---------|------|
| `provider.tf` | Google + Elastic Cloud provider（credentials.yml を yamldecode） |
| `variables.tf` | ルート変数 |
| `main.tf` | モジュール呼び出し（依存グラフ明示） |
| `outputs.tf` | ルート出力（API URL・ESエンドポイント・SA email） |
| `terraform.tfvars` | Elastic Cloud APIキー ※git管理外 |
| `modules/data/` | データ層: GCS + BigQuery dataset/table |
| `modules/compute/` | コンピュート層: Cloud Run Job/Service + Scheduler |
| `modules/elastic/` | 外部SaaS: Elastic Cloud + Secret Manager |
| `modules/registry/` | コンテナレジストリ: Artifact Registry |
| `modules/iam/` | 認証認可: SA + 最小権限 IAM バインディング |
