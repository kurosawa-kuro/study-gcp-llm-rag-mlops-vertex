## ディレクトリ構成

```
study-gcp-llm-rag-mlops-vertex/
├── .github/workflows/          # CI/CD（GitHub Actions）
├── data/sample/                # サンプル社内規定ドキュメント
├── docs/                       # 仕様・設計書・運用マニュアル
├── env/
│   ├── config/                 # プロジェクト設定（application.yml）
│   └── secret/                 # クレデンシャル（git管理外）
├── scripts/
│   ├── monitor/                # 監視スクリプト
│   ├── ops/                    # 操作スクリプト
│   └── setup/                  # 環境構築スクリプト
├── shared/                     # 共通ライブラリ（config + core）
├── src/doc-qa/
│   ├── api/                    # Cloud Run Service（QA API）
│   │   ├── endpoints/          #   FastAPIルーティング
│   │   ├── search/             #   検索・リランク
│   │   ├── generation/         #   LLM回答生成
│   │   └── tests/              #   テスト
│   └── ingestion/              # Cloud Run Job（ドキュメント取込）
│       ├── extract/            #   テキスト抽出・チャンク分割
│       ├── embed/              #   ベクトル化
│       ├── store/              #   格納（BQ + ES）
│       └── tests/              #   テスト
├── terraform/
│   └── modules/                # data / compute / elastic / registry / iam
└── Makefile                    # 全操作の統合エントリポイント
```

### src/doc-qa/ingestion/（ドキュメント取込パイプライン）

| ファイル | 説明 |
|---------|------|
| `main.py` | オーケストレーション（shared/config.py 経由で全設定取得） |
| `extract/extractor.py` | テキスト抽出（PDF: pymupdf / Word: python-docx / TXT） |
| `extract/chunker.py` | チャンク分割（引数で chunk_size/overlap を受け取る） |
| `embed/embedder.py` | Vertex AI Embedding API（引数で model_name/batch_size を受け取る） |
| `store/bq_writer.py` | BigQuery documents テーブル書き込み（冪等・リトライ付き） |
| `store/es_writer.py` | Elasticsearch kuromoji インデックス登録（引数で index_name を受け取る） |
| `tests/test_main.py` | 単体テスト（extractor, chunker, bq_writer） |
| `Dockerfile` | build context=ルート、shared/config.py + application.yml を含む |
| `requirements.txt` | 依存パッケージ |
| `requirements-dev.txt` | 開発依存（pytest, pytest-mock） |

| `requirements-dev.txt` | 開発依存（pytest, httpx, pytest-mock） |

### terraform/（モジュール分離構成）

| ファイル | 説明 |
|---------|------|
| `provider.tf` | Google + Elastic Cloud provider（credentials.yml を yamldecode） |
| `variables.tf` | ルート変数（プロジェクト・リージョン・各種名前・ES認証情報） |
| `main.tf` | モジュール呼び出し（依存グラフ: registry→data→elastic→iam→compute） |
| `outputs.tf` | ルート出力（API URL・ESエンドポイント・SA email） |
| `terraform.tfvars` | Elastic Cloud APIキー ※git管理外 |
| `modules/data/` | データ層: GCS + BigQuery dataset/table（各 main/variables/outputs.tf） |
| `modules/compute/` | コンピュート層: Cloud Run Job/Service + Scheduler + Job invoker IAM |
| `modules/elastic/` | 外部SaaS: Elastic Cloud デプロイメント + Secret Manager |
| `modules/registry/` | コンテナレジストリ: Artifact Registry |
| `modules/iam/` | 認証認可: SA + BQ/GCS/VertexAI/SecretManager 権限バインディング |
