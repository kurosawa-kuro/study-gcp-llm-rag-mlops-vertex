### src/doc-qa/ingestion/（ドキュメント取込パイプライン）

| ファイル | 説明 |
|---------|------|
| `main.py` | オーケストレーション（GCS取得→抽出→分割→Embedding→BQ/ES格納） |

| `chunker.py` | チャンク分割（800文字・50文字オーバーラップ） |
| `embedder.py` | Vertex AI Embedding API呼び出し（バッチ100件/リクエスト） |
| `bq_writer.py` | BigQuery documents テーブル書き込み（冪等・リトライ付き） |
| `es_writer.py` | Elasticsearch kuromoji インデックス登録（冪等） |
| `Dockerfile` | Python 3.11-slim コンテナ定義 |
| `requirements.txt` | 依存パッケージ |
| `requirements-dev.txt` | 開発依存（pytest, pytest-mock） |
| `test_main.py` | 単体テスト（extractor, chunker, bq_writer） |
