### 検証時に発覚し修正した問題

| 問題 | 原因 | 修正 |
|------|------|------|
| Cloud Run Service 起動失敗 | `__init__.py` 未作成でサブフォルダが Python パッケージとして認識されない | 全サブフォルダに `__init__.py` 追加 |
| BQ 書き込みエラー | ストリーミングバッファ中の行に DELETE 不可 | `bq_writer.py` で streaming buffer エラー時はスキップ |
| ES インデックス作成失敗 | kuromoji アナライザー未インストール | `standard` アナライザーに変更 |
| Gemini 404 (Vertex AI) | コンソールで利用規約未同意 + モデル名廃止 | Google AI Studio SDK に切替 + `gemini-2.5-flash` |
| ES 認証エラー 401 | Secret Manager に旧デプロイメントの API キーが格納 | `ec_deployment` から username/password を自動取得して格納 |
| Docker イメージ未更新 | Docker キャッシュで変更未反映 | `--no-cache` でリビルド |
| `report.py` 空リストで TypeError | `generate_report([])` で `max()` に空シーケンスを展開 | 早期リターンでメッセージを返すよう修正 |
| API Service Terraform 変数未定義 | `compute/main.tf` で `var.gemini_api_key`（存在しない変数）を参照 | `var.google_ai_studio_api_key` に修正 |
| eval イメージ未 push で tf-apply 失敗 | bootstrap に eval-pipeline-push が漏れていた | `images-push` 集約ターゲットで構造的に防止 |
| API 起動失敗（ES 未準備） | Terraform が elastic 完了前に compute を開始 | `depends_on = [module.elastic]` + アプリ側 ES 接続リトライ（最大3分） |
| `tf-destroy` 対話入力待ち | `-auto-approve` 未指定 | 追加 |
| BQ Vector Index 作成失敗 | `index_type` 未指定 / テーブル空 | `IVF` 指定 + ingestion 後に移動 + 5000行未満スキップ |
| Gemini API レートリミットで評価中断 | 無料枠 日次20回/分次5回 | evaluate.py にリトライロジック追加（最大5回） |
